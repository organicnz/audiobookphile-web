# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to tamerlanium@gmail.com. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:
- Type of issue (e.g., cross-site scripting, SQL injection, remote code execution)
- The location of the affected code or endpoint
- Step-by-step instructions to reproduce the issue
- Any potential impact of the vulnerability

## Supported Versions

We only provide security updates for the latest stable version of the application.

## Acknowledgements

We appreciate the work of security researchers who help us keep our users safe.
